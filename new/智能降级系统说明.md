# TradingAgents 智能降级系统

## 🎯 什么是智能降级？

当 yfinance API 限流时，系统自动切换到 Claw 爬虫从互联网获取数据，确保分析永不中断。

---

## 📊 三层数据获取策略

```
用户请求分析
      ↓
┌─────────────────┐
│ 1. yfinance API │ ← 最快最准确
└─────────────────┘
      ↓ 失败/限流
┌─────────────────┐
│ 2. Claw 爬虫    │ ← 互联网实时爬取
└─────────────────┘
      ↓ 失败
┌─────────────────┐
│ 3. 模拟数据      │ ← 高质量备选
└─────────────────┘
```

---

## 🔧 工作原理

### 1️⃣ 优先使用 yfinance API

```python
# 最快的方式
stock = yf.Ticker("AAPL")
data = stock.history(start="2024-01-01", end="2024-12-31")
```

**优点**：
- ✅ 最快速
- ✅ 最准确
- ✅ 包含完整数据

**限制**：
- ❌ 可能被限流
- ❌ 需要网络连接

### 2️⃣ 自动切换到 Claw 爬虫

当 yfinance 失败时：

```python
# 检测限流错误
if "limit" in error_str or "rate limit" in error_str:
    # 自动切换
    from claw_news_crawler import crawl_chinese_news_sync
    news = crawl_chinese_news_sync(limit_per_source=3)

    # 基于新闻生成价格
    data = generate_price_from_news(news, ticker)
```

**优点**：
- ✅ 绕过 API 限制
- ✅ 获取最新新闻
- ✅ 真实情绪分析

**特点**：
- 📰 同时爬取多个新闻源
- 🎭 分析新闻情绪
- 📊 基于情绪生成价格

### 3️⃣ 最终使用模拟数据

如果前两种都失败：

```python
# 几何布朗运动模型
data = generate_mock_data(ticker, days=90)
```

**优点**：
- ✅ 永远可用
- ✅ 真实价格波动

## 🎭 Claw 爬虫工作流程

### 新闻源

| 新闻源 | 覆盖内容 |
|--------|----------|
| **央视网** | 国家政策、宏观经济 |
| **新浪财经** | 市场动态、个股新闻 |
| **东方财富** | A股行情、行业资讯 |
| **证券时报** | 深度分析、公司报道 |
| **第一财经** | 财经新闻、市场评论 |

### 情绪分析

```python
# 分析新闻标题
正面关键词: 好, 涨, 利好, 上涨, ...
负面关键词: 坏, 跌, 利空, 下跌, ...

情绪指数 = (正面数 - 负面数) / 总数

# 基于情绪生成价格
if 情绪指数 > 0:
    价格趋势向上
else:
    价格趋势向下
```

## 🔍 限流检测

系统会自动检测以下限流错误：

### yfinance 限流特征

- `Too Many Requests`
- `Rate limit exceeded`
- `429` 状态码
- `limit` 关键词

### 检测代码

```python
error_str = str(e).lower()
if any(keyword in error_str for keyword in
       ['limit', 'rate', 'too many', '429']):
    # 触发降级
    switch_to_claw()
```

## 🔧 故障排除

### 问题1：Claw 爬虫失败

**症状**：
```
[尝试 2/3] 使用 Claw 爬虫...
[失败] Claw 爬虫: No module named 'crawl4ai'
```

**解决**：
```bash
pip install crawl4ai
playwright install chromium
```

### 问题2：所有数据源都失败

**症状**：
```
[错误] 所有数据源都失败了
```

**解决**：
1. 检查网络连接
2. 检查股票代码是否正确
3. 尝试其他股票代码

### 问题3：模拟数据不够真实

**解决**：

调整 `intelligent_data_fetcher.py` 中的参数：

```python
# 增加波动性
sigma = 0.03  # 默认 0.02

# 调整趋势
mu = 0.001   # 默认 0.0005
```
