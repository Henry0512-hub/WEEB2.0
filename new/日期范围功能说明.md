# TradingAgents 日期范围功能说明

## ✅ 新增功能：日期范围输入

现在系统支持输入**开始日期**和**结束日期**，可以灵活指定分析的时间范围！

---

## 🎯 功能特点

### 1. 灵活的日期范围
```
开始日期: 2025-12-23
结束日期: 2026-02-13
```
系统将分析这个日期范围内的股票表现。

### 2. 自动数据源选择
```
开始日期 ≤ 2024-12-31
  ↓
使用 WRDS 学术数据库

开始日期 > 2024-12-31
  ↓
使用实时数据 + 智能降级
```

### 3. 智能降级保障
```
第1层: yfinance API
   ↓ 失败/限流
第2层: Claw 爬虫
   ↓ 失败
第3层: 模拟数据
   ↓
100% 可用
```

## 💻 命令行使用

### 基本格式

```bash
python run_enhanced_analysis.py <股票代码> <开始日期> <结束日期> <分析师> <类型>
```

### 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| 股票代码 | AAPL, TSLA, NVDA 等 | AAPL |
| 开始日期 | YYYY-MM-DD 格式 | 2025-01-15 |
| 结束日期 | YYYY-MM-DD 格式 | 2025-03-20 |
| 分析师 | 1=DeepSeek, 2=Kimi, 3=Gemini | 1 |
| 分析类型 | 1=完整, 2=快速, 3=情绪 | 1 |

### 实际命令示例

```bash
# 示例1: 分析AAPL的2个月数据（使用DeepSeek）
python run_enhanced_analysis.py AAPL 2025-01-15 2025-03-20 1 1

# 示例2: 分析TSLA的1个月数据（使用Kimi）
python run_enhanced_analysis.py TSLA 2025-02-01 2025-03-01 2 1

# 示例3: 分析NVDA的3个月数据（使用Gemini）
python run_enhanced_analysis.py NVDA 2025-01-01 2025-04-01 3 1

# 示例4: 分析历史数据（自动使用WRDS）
python run_enhanced_analysis.py AAPL 2024-06-15 2024-08-15 1 1

# 示例5: 情绪分析
python run_enhanced_analysis.py TSLA 2025-03-01 2025-04-09 1 3
```

## ⚠️ 注意事项

### 1. 日期格式

```
✅ 正确: 2025-01-15
❌ 错误: 01-15-2025
❌ 错误: 2025/01/15
❌ 错误: 20250115
```

### 2. 开始日期 ≤ 结束日期

```
✅ 正确:
   开始: 2025-01-15
   结束: 2025-03-20

❌ 错误:
   开始: 2025-03-20
   结束: 2025-01-15
```

### 3. 结束日期可以为空

```
开始日期: 2025-01-15
结束日期: (直接回车)

系统自动使用今天作为结束日期
```

### 4. 历史数据限制

```
开始日期 ≤ 2024-12-31
  ↓ 自动使用 WRDS

开始日期 > 2024-12-31
  ↓ 使用实时数据
```
