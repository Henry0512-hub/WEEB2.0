### 2. 自动重连机制
```python
def get_wrds_connection():
    """获取WRDS连接（自动重连）"""
    global _wrds_connection
    if _wrds_connection is None or not _wrds_connection._connected:
        _wrds_connection = WRDSConnection()
        _wrds_connection.connect()
    return _wrds_connection.db
```
