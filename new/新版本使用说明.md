# TradingAgents 新版使用说明

## 🎯 重新设计的分析流程

### 新流程特点

✅ **第一步：选择AI分析师** (3选1)
- DeepSeek (最便宜)
- Kimi (中文最好)
- Gemini (免费)

✅ **第二步：输入股票代码**
- 美股：AAPL, TSLA, NVDA等
- 中概股：BABA, JD, PDD等
- 加密货币：BTC-USD, ETH-USD

✅ **第三步：选择分析日期**
- **2024-12-31及以前** → 自动使用WRDS学术数据库
- **2024-12-31以后** → 自动使用实时新闻+基本面

✅ **第四步：选择分析类型**
- 完整分析（技术+基本面+情绪）
- 快速分析（技术+基本面）
- 情绪分析（新闻情感）

## 🔧 增强版分析脚本

新脚本 `run_enhanced_analysis.py` 包含：

### 1. 智能数据源选择
```python
def should_use_wrds(date_str):
    """根据日期自动选择数据源"""
    if date <= "2024-12-31":
        使用 WRDS + 新闻
    else:
        使用 实时新闻 + 基本面
```

### 2. 三位分析师配置
```python
分析师1: DeepSeek
  - API: https://api.deepseek.com
  - 价格: ¥1/百万tokens
  - 模型: deepseek-chat

分析师2: Kimi
  - API: https://api.moonshot.cn
  - 优势: 128k上下文
  - 模型: moonshot-v1-8k

分析师3: Gemini
  - API: https://ai.google.dev
  - 价格: 免费1500次/天
  - 模型: gemini-2.5-flash
```

### 3. 三种分析模式

**完整分析** (type=1)
- 技术面：技术指标、趋势分析
- 基本面：财务数据、估值分析
- 情绪面：新闻情感、市场情绪
- 辩论轮数：2轮（深度分析）

**快速分析** (type=2)
- 技术面分析
- 基本面分析
- 辩论轮数：1轮（快速输出）

**情绪分析** (type=3)
- 新闻情感分析
- 市场情绪评估
- 辩论轮数：1轮（专注情绪）

## ⚙️ 高级配置

### 自定义日期范围

```python
# 历史分析（使用WRDS）
date = "2024-06-15"

# 近期分析（使用实时数据）
date = "2025-01-15"

# 今天分析（使用实时数据）
date = ""  # 直接回车
```

### 调整分析深度

在 `run_enhanced_analysis.py` 中修改：

```python
# 完整分析：增加深度
trade_config["max_debate_rounds"] = 2  # 辩论轮数
trade_config["max_risk_discuss_rounds"] = 2

# 快速分析：减少轮数
trade_config["max_debate_rounds"] = 1
trade_config["max_risk_discuss_rounds"] = 1
```
