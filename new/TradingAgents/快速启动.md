## ❓ 常见问题

### Q1: 双击没反应？
**A:** 确保已安装 Python 3.10+，并在 TradingAgents 目录下运行

### Q2: 提示找不到模块？
**A:** 运行 `cd TradingAgents && pip install -e .`

### Q3: API 连接失败？
**A:** 运行测试脚本检查 API Key：`python test_deepseek_simple.py`

### Q4: 想要更快的结果？
**A:** 修改配置减少辩论轮数：
```python
config["max_debate_rounds"] = 1
```
