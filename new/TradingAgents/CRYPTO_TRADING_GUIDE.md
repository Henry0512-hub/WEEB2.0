# TradingAgents 加密货币交易推荐系统 使用指南

## 🎯 系统概述

TradingAgents 是一个基于多智能体AI的加密货币交易推荐系统，通过8个专业AI智能体协同分析，为你提供**清晰、可操作的交易建议**。

**⚠️ 重要提示：本系统仅提供交易建议，不执行实际交易。所有交易决策由你自行决定和执行。**

---

## 🚀 快速开始

### 方式1：加密货币专用模式（推荐）

```bash
# 运行加密货币专用分析脚本
python run_crypto_trading.py
```

### 方式2：使用启动脚本

双击桌面上的 `启动TradingAgents加密货币版.bat`，选择 **选项1 - 加密货币分析模式**

### 方式3：通用分析模式

```bash
# 支持股票、A股、加密货币混合分析
python run_with_deepseek_loop.py
```

## 📊 交易推荐评级

系统会给出以下5种推荐之一：

| 评级 | 英文 | 含义 | 操作建议 |
|------|------|------|----------|
| 🟢 | **Buy** | 强力买入 | 建立新仓位或加仓 |
| 🟢 | **Overweight** | 增持 | 逐步增加仓位 |
| 🟡 | **Hold** | 持有 | 维持当前仓位，观望 |
| 🟠 | **Underweight** | 减持 | 减少仓位，部分止盈 |
| 🔴 | **Sell** | 卖出 | 清仓或避免建仓 |

## 📋 推荐报告内容

每次分析后，你会收到包含以下内容的报告：

### 1. **推荐评级**
- 明确的买入/持有/卖出建议
- 中文含义说明

### 2. **执行摘要**
- 入场策略建议
- 仓位大小建议
- 关键风险价位
- 目标价位
- 时间周期

### 3. **投资论点**
- 详细的分析逻辑
- 技术面分析
- 基本面分析
- 市场情绪分析

### 4. **风险提示**
- 潜在风险因素
- 止损建议

## ⚠️ 重要风险提示

1. **仅供参考** - AI推荐仅供参考，不构成投资建议
2. **自行决策** - 所有交易决策由你自行负责
3. **市场风险** - 加密货币市场波动极大，可能损失全部本金
4. **技术限制** - AI可能产生错误判断，请结合其他分析
5. **资金管理** - 建议只用可承受损失的资金交易
6. **分散投资** - 不要将所有资金投入单一资产

### 建议的风险管理原则

- ✅ 单笔交易不超过总资金的2-5%
- ✅ 设置止损位（建议-5%到-10%）
- ✅ 设置止盈位（建议+10%到+20%）
- ✅ 不要满仓操作
- ✅ 分散投资3-5个不同资产
- ✅ 保持冷静，不要情绪化交易

## 🛠️ 故障排除

### 问题1：CoinGecko API连接超时

**解决方案**：
- 检查网络连接
- 如在中国大陆，可能需要配置代理
- 系统会自动fallback到其他数据源

### 问题2：API余额不足

**解决方案**：
- 检查DeepSeek API余额
- 减少辩论轮数以降低API调用
- 使用更便宜的模型（如deepseek-chat）

### 问题3：分析时间过长

**解决方案**：
- 降低`max_debate_rounds`和`max_risk_discuss_rounds`
- 使用更快的模型（如gpt-3.5-turbo）

## 🎓 最佳实践

### ✅ 推荐做法

1. **多币种分析** - 不要只分析一个币种，对比多个机会
2. **定期更新** - 市场变化快，定期重新分析
3. **记录决策** - 保存每次分析的信号，便于复盘
4. **结合实际** - 将AI建议与自己的判断结合
5. **小仓位试验** - 先用小仓位测试系统建议的准确性

### ❌ 避免做法

1. **盲目跟随** - 不要完全依赖AI建议
2. **频繁交易** - 避免过度交易增加成本
3. **追涨杀跌** - 保持冷静，不要情绪化
4. **孤注一掷** - 绝不要All-in单一资产
5. **忽视风险** - 始终设置止损
