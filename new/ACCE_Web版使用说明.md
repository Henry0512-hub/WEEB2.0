# 🌐 ACCE v2.0 - Web界面使用说明

## ✨ 功能特点

### 🎨 后现代主义设计
- **霓虹灯风格** - 青色和洋红色的霓虹灯效果
- **动态背景** - 渐变动画和网格叠加层
- **响应式布局** - 适配各种屏幕尺寸
- **毛玻璃效果** - 卡片使用 backdrop-filter 模糊效果

### 🤖 AI智能体工作流程
- **8个AI智能体** - 实时显示工作状态
- **霓虹灯节点** - 工作中的智能体会闪烁发光
- **进度条** - 显示整体分析进度
- **状态更新** - 每个智能体实时更新状态

### 📊 图表展示
- **价格走势图** - 使用 Chart.js 绘制折线图
- **技术指标图** - 柱状图显示 RSI、MACD、MA 等指标
- **实时更新** - 分析过程中图表动态更新

### 📝 分析报告
- **投资建议** - 买入/卖出/持有推荐
- **技术分析** - RSI、MACD、移动平均线
- **基本面分析** - P/E比率、营收增长
- **情绪分析** - 新闻情绪、社交媒体情绪
- **风险评估** - 止损价、目标价、时间周期

### 💻 终端输出
- **实时日志** - 显示系统运行日志
- **错误信息** - 高亮显示错误信息
- **自动滚动** - 自动滚动到最新输出

## 🚀 使用方法

### 方法1：使用启动器（推荐）

1. **双击启动器**
   ```
   C:\Users\lenovo\Desktop\启动ACCE_Web版.bat
   ```

2. **等待启动**
   - 启动器会自动安装依赖
   - 浏览器会自动打开 http://localhost:5000

3. **配置参数**
   - 选择AI分析师（DeepSeek、Kimi、Gemini）
   - 输入股票代码（AAPL、TSLA、NVDA等）
   - 选择日期范围
   - 选择分析类型

4. **开始分析**
   - 点击"🚀 开始分析"按钮
   - 观察工作流程图实时更新
   - 查看图表动态变化
   - 阅读最终分析报告

### 方法2：手动启动

1. **安装依赖**
   ```bash
   cd C:\Users\lenovo\TradingAgents
   python -m pip install flask flask-sock flask-cors
   ```

2. **启动服务器**
   ```bash
   python web_backend.py
   ```

3. **打开浏览器**
   - 访问 http://localhost:5000
