## 🎯 使用示例

### 基础使用

```python
from tradingagents.graph.trading_graph import TradingAgentsGraph
from tradingagents.default_config import DEFAULT_CONFIG

# 使用 DeepSeek
config = DEFAULT_CONFIG.copy()
config["llm_provider"] = "deepseek"
config["backend_url"] = "https://api.deepseek.com/v1"
config["deep_think_llm"] = "deepseek-chat"
config["output_language"] = "Chinese"

ta = TradingAgentsGraph(debug=True, config=config)
_, decision = ta.propagate("AAPL", "2025-01-15")
print(decision)
```

### 切换 API

```python
# 切换到 Kimi
config["llm_provider"] = "openai"
config["backend_url"] = "https://api.moonshot.cn/v1"
config["deep_think_llm"] = "moonshot-v1-8k"
import os
os.environ["OPENAI_API_KEY"] = "your-kimi-key"

# 切换到 Gemini
config["llm_provider"] = "google"
config["deep_think_llm"] = "gemini-2.5-flash"
```

## 💡 实用技巧

### 降低成本

```python
# 减少辩论轮数
config["max_debate_rounds"] = 1
config["max_risk_discuss_rounds"] = 1

# 使用更便宜的模型
config["deep_think_llm"] = "deepseek-chat"  # 最便宜
```

### 提高质量

```python
# 增加辩论轮数
config["max_debate_rounds"] = 2
config["max_risk_discuss_rounds"] = 2

# 使用更强的模型
config["deep_think_llm"] = "deepseek-reasoner"
config["deep_think_llm"] = "gemini-2.5-pro"
config["deep_think_llm"] = "moonshot-v1-128k"
```

### 切换语言

```python
config["output_language"] = "Chinese"  # 中文
config["output_language"] = "English"  # 英文
```
