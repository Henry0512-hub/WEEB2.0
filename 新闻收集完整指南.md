## 🔧 配置示例

### 使用 yfinance（默认）
```python
from tradingagents.default_config import DEFAULT_CONFIG

config = DEFAULT_CONFIG.copy()

# 使用 yfinance 获取新闻（默认）
config["data_vendors"]["news_data"] = "yfinance"

# 运行
ta = TradingAgentsGraph(debug=True, config=config)
ta.propagate("AAPL", "2025-01-15")
```

### 使用 Alpha Vantage（可选）
```python
# 设置 API Key
import os
os.environ["ALPHA_VANTAGE_API_KEY"] = "your-key"

# 配置
config["data_vendors"]["news_data"] = "alpha_vantage"

# 运行
ta = TradingAgentsGraph(debug=True, config=config)
ta.propagate("AAPL", "2025-01-15")
```

### 使用 akshare（中国股票）
```python
from tradingagents.dataflows.akshare_news import get_chinese_stock_news

# 获取 A 股新闻
news = get_chinese_stock_news("000001")
print(news)
```
