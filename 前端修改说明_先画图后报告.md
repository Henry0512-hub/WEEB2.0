# ACCE v2.0 - 前端修改说明文档

**优化目标**: 先显示图表（3-5秒），后端AI分析在后台进行，流式显示分析报告

---

## 📊 新的API端点

### 1. `/api/chart-data` (POST) - 快速获取图表数据

**请求示例**:
```javascript
const response = await fetch('/api/chart-data', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        ticker: 'AAPL',
        start_date: '2024-06-01',
        end_date: '2024-08-15'
    })
});

const result = await response.json();
// result.data 包含:
// - ohlcv: K线数据
// - indicators: 技术指标 (MA, RSI, MACD, KDJ, BOLL)
// - summary: 当前价格信息
```

**响应时间**: 3-5秒（从WRDS获取准确数据）

---

### 2. `/api/analyze` (POST) - 启动AI分析（立即返回）

**请求示例**:
```javascript
const response = await fetch('/api/analyze', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        llm_choice: '1',          // 1=DeepSeek, 2=Kimi, 3=Gemini
        ticker: 'AAPL',
        start_date: '2024-06-01',
        end_date: '2024-08-15',
        analysis_type: '1',       // 1=完整, 2=快速, 3=情绪
        market_type: 'us'         // us, cn, crypto
    })
});

const result = await response.json();
// result.task_id: 任务ID，用于后续查询进度
```

**响应时间**: <1秒（立即返回task_id）

---

### 3. `/api/task/<task_id>/progress` (GET) - SSE流式获取AI进度

**使用示例**:
```javascript
const eventSource = new EventSource(`/api/task/${taskId}/progress`);

eventSource.onmessage = (event) => {
    const data = JSON.parse(event.data);

    if (data.type === 'done') {
        console.log('AI分析完成');
        eventSource.close();
        return;
    }

    if (data.agent) {
        // 更新智能体状态
        updateAgentStatus(data.agent, data.stage);
    }

    if (data.text) {
        // 显示AI输出
        appendLog(data.text);
    }
};

eventSource.onerror = (error) => {
    console.error('SSE错误:', error);
    eventSource.close();
};
```

**特点**:
- 流式传输，实时显示进度
- 自动重连
- 服务器推送更新

---

### 4. `/api/task/<task_id>` (GET) - 查询任务状态

**请求示例**:

```javascript
const response = await fetch(`/api/task/${taskId}`);
```

## 🔄 新的前端工作流程

### 旧流程（慢）
```
1. 用户点击"开始分析"
2. 显示"加载中..."（白屏）
3. 等待5-8分钟 ⚠️
4. 突然显示所有内容（图表+报告）
```

### 新流程（快）⚡
```
1. 用户点击"开始分析"

2. 并行执行：
   ├─► A. 立即获取图表数据
   │   └─► 3-5秒后显示K线图和指标线 ✅
   │
   └─► B. 启动AI分析（后台）
       └─► 立即返回task_id
           └─► 使用SSE流式显示AI报告

3. 用户体验：
   - 5秒后看到图表（可以研究、缩放）
   - AI报告逐段显示（有进度感）
   - 无白屏焦虑
```

```javascript
function renderChart(chartData) {
    const {ohlcv, indicators} = chartData.data;

    // 准备数据
    const dates = ohlcv.date;
    const klineData = ohlcv.date.map((date, i) => [
        ohlcv.open[i],
        ohlcv.close[i],
        ohlcv.low[i],
        ohlcv.high[i]
    ]);

    // 初始化ECharts
    const chart = echarts.init(document.getElementById('chart-container'));

    const option = {
        title: { text: `${chartData.ticker} - 日K线` },
        tooltip: { trigger: 'axis' },
        legend: { data: ['K线', 'MA5', 'MA10', 'MA20', 'MA60'] },
        grid: [
            { left: '10%', right: '8%', height: '50%' },   // K线图
            { left: '10%', right: '8%', top: '63%', height: '16%' },  // 成交量
            { left: '10%', right: '8%', top: '82%', height: '14%' }   // MACD
        ],
        xAxis: [
            { type: 'category', data: dates, gridIndex: 0, axisLabel: {show: false} },
            { type: 'category', data: dates, gridIndex: 1, axisLabel: {show: false} },
            { type: 'category', data: dates, gridIndex: 2 }
        ],
        yAxis: [
            { scale: true, gridIndex: 0, splitLine: {show: true} },  // K线
            { scale: true, gridIndex: 1, splitLine: {show: false} },  // 成交量
            { scale: true, gridIndex: 2, splitLine: {show: false} }   // MACD
        ],
        dataZoom: [
            { type: 'inside', xAxisIndex: [0, 1, 2], start: 50, end: 100 },
            { show: true, xAxisIndex: [0, 1, 2], type: 'slider', top: '95%', start: 50, end: 100 }
        ],
        series: [
            {
                name: 'K线',
                type: 'candlestick',
                data: klineData,
                itemStyle: {
                    color: '#ef232a',      // 阳线颜色（涨）
                    color0: '#14b143',     // 阴线颜色（跌）
                    borderColor: '#ef232a',
                    borderColor0: '#14b143'
                }
            },
            {
                name: 'MA5', type: 'line', data: indicators.MA5, smooth: true,
                lineStyle: {opacity: 0.8}
            },
            {
                name: 'MA10', type: 'line', data: indicators.MA10, smooth: true,
                lineStyle: {opacity: 0.8}
            },
            {
                name: 'MA20', type: 'line', data: indicators.MA20, smooth: true,
                lineStyle: {opacity: 0.8}
            },
            {
                name: 'MA60', type: 'line', data: indicators.MA60, smooth: true,
                lineStyle: {opacity: 0.8}
            },
            {
                name: 'Volume',
                type: 'bar',
                xAxisIndex: 1,
                yAxisIndex: 1,
                data: ohlcv.volume,
                itemStyle: {
                    color: function(params) {
                        return klineData[params.dataIndex][1] > klineData[params.dataIndex][0]
                            ? '#ef232a' : '#14b143';
                    }
                }
            },
            {
                name: 'MACD',
                type: 'bar',
                xAxisIndex: 2,
                yAxisIndex: 2,
                data: indicators.MACD.MACD,
                itemStyle: {
                    color: function(params) {
                        return params.data >= 0 ? '#ef232a' : '#14b143';
                    }
                }
            }
        ]
    };

    chart.setOption(option);
}
```
