# ACC102 Track 4 — Demo video checklist (1–3 minutes, English)

Use this when recording the **required** demo. Narration or **English subtitles** are mandatory per the brief.

## Suggested structure

| Time | Content |
|------|---------|
| 0:00–0:20 | Title: product name, your name/ID, Track 4. |
| 0:20–0:50 | Show environment: terminal, `pip install -r requirements_track4.txt`, `python web_backend.py`. |
| 0:50–1:40 | Browser: open `http://127.0.0.1:5000`, enter ticker, dates, market type, start analysis; show chart + report/news panels loading. |
| 1:40–2:30 | Open `notebooks/ACC102_Track4_Analytical_Workflow.ipynb` in Jupyter: run one cell that loads data or show pre-run outputs + one chart. |
| 2:30–3:00 | One sentence: data source (WRDS) + disclaimer (educational, not advice). |

## Technical tips

- **Resolution:** 1280×720 or 1080p; readable font size in terminal and browser.  
- **Audio:** Text-to-speech is allowed; ensure captions/subtitles match.  
- **Secrets:** Do not show API keys or WRDS passwords; blur if needed.  
- **Hosting:** Upload to YouTube (unlisted), OneDrive, or your institution’s platform; submit **LMO-accessible** link.
