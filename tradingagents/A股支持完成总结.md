# ✅ A股支持已集成！

## 🎉 新功能

现在TradingAgents已经支持**中国A股**分析！

---

## 📊 支持的股票类型

### ✅ A股（新功能）
- **格式：** `代码.SS` 或 `代码.SZ`（如：600519.SS, 000001.SZ）
- **数据源：** EFinance（专门的中国金融数据源）
- **自动检测：** 系统会自动识别A股代码并使用EFinance

### ✅ 美股
- **格式：** 直接代码（如：AAPL, TSLA）
- **数据源：** yfinance

### ✅ 中概股
- **格式：** 直接代码（如：BABA, JD, PDD）
- **数据源：** yfinance

### ✅ 港股
- **格式：** 代码.HK（如：0700.HK, 9988.HK）
- **数据源：** yfinance

---

## 🚀 使用方式

### 方式1：桌面快捷方式（推荐）

1. 双击桌面上的 `启动TradingAgents修复版.bat`
2. 选择 `1`（交互式分析）
3. 输入A股代码，例如：
   - `600519.SS` - 贵州茅台
   - `000001.SZ` - 平安银行
   - `600036.SS` - 招商银行

### 方式2：命令行

```bash
cd C:\Users\lenovo\TradingAgents
python run_with_deepseek_interactive.py
```
