在 CLI 中：
1. 选择股票代码（如：BABA, 600519.SS）
2. 选择分析日期
3. 选择 LLM Provider（选择 DeepSeek）
4. 等待分析完成

### 方式 3：在你的代码中使用

```python
from tradingagents.graph.trading_graph import TradingAgentsGraph
from tradingagents.default_config import DEFAULT_CONFIG
import os

# 设置 API
os.environ["OPENAI_API_KEY"] = "sk-d28ae30a58cb496c9b40e0029d0ef2c1"

# 配置 DeepSeek
config = DEFAULT_CONFIG.copy()
config["llm_provider"] = "deepseek"
config["backend_url"] = "https://api.deepseek.com/v1"
config["deep_think_llm"] = "deepseek-chat"
config["quick_think_llm"] = "deepseek-chat"
config["output_language"] = "Chinese"

# 创建实例并运行
ta = TradingAgentsGraph(debug=True, config=config)
_, decision = ta.propagate("BABA", "2025-01-15")
print(decision)
```

## ❓ 常见问题

### Q: 如何查看 API 使用情况？
A: 登录 DeepSeek 官网查看账户余额和使用记录

### Q: 分析失败怎么办？
A:
1. 检查网络连接
2. 确认 API 密钥有效
3. 尝试更换股票代码或日期
4. 查看错误日志

### Q: 支持中国股市吗？
A: 部分支持。yfinance 对中国股市数据有限制，可以集成 akshare 获取更好的中国数据

### Q: 如何集成 akshare？
A: 可以创建 akshare 数据适配器，或者直接在代码中使用 akshare 获取数据
